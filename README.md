# Profil Utilisateur IRIS

## Description
Page de profil utilisateur réalisée dans le cadre du workshop HTML Masterclass d'IRIS Junior Création. Elle présente un profil sémantique, accessible et conforme aux normes W3C.

## Technologies utilisées
- HTML5
- Git / GitHub

## Fonctionnalités
- En-tête avec avatar et navigation interne
- Section À propos
- Section Compétences
- Formulaire de contact
- Pied de page avec liens réseaux sociaux
